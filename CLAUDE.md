## 0. このプロジェクトとは

**Metalogos**（メタロゴス）は、karasuship が個人事業主として運営するサービス群の統合ブランド。

「やるかやらないかだけの状態にして、申請・記録・申告まで全部出す」

ユーザーはGOを押すだけ。Metalogosが手順・申請書・記録・経費計算・確定申告書類を出す。

## 4. tax-nav の設計思想

### 何を作るのか
個人事業主（青色申告）として活動するユーザーのための節税・申告ナビ。

Metalogosの他サービスと連動する：
- 賃貸チェッカーで確認行動 → 経費として記録
- ポイ活・せどり・分散ネットワーク → 収入として記録
- 全部まとめて → 青色申告65万控除まで案内

### ユーザーが入力すること（最小）
- 活動の種類（ポイ活・せどり・副業など）
- 収入・経費の記録
- 自宅の使用状況（按分計算用）

### Metalogosが出すもの
- 経費按分の計算（自宅・通信費・デバイス）
- 扶養判定（103万・130万・150万の壁）
- 青色申告用の帳簿データ
- 確定申告書類の下書き
- やるべきことを1つずつ出すナビゲーション

## 5. 認証方針

新しいサービスからはSupabase認証を最初から組み込む。
`apps/tax-nav/lib/supabase.ts` にクライアントを配置済み。

.env.localの場所：`apps/tax-nav/.env.local`（gitignore済み）
必要な環境変数：
- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`

## 7. 生態系の全体像

```
賃貸費用チェッカー → 確認行動の実態
せどりツール       → 売買事業の実態
分散ネットワーク    → 計算資源提供事業の実態
節税ツール（tax-nav）→ 上記全部の経費計算・扶養調整
QRIP             → 上記で得た収益の投資判断
```

報酬の循環：
分散ネットワーク貢献 → AIクレジット（サービス内）→ せどり・仮説検証に使う → 実際の収入 → 節税ツールで経費を落とす
